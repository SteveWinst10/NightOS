#NightOS
A WebOS with a dark and glassy theme made using pure html, css and js.

Try it out here:https://stevewinst10.github.io/NightOS/

<span style="color:yellow; font-size:30px; font-weight:bold;">Note:</span>
Project is still in very early stage.

##Features:
* **UI/UX :** A minimalistic dark and glassy theme.
* **Taskbar :** Dynamic taskbar with changable positions.(Not done yet)
* **Settings :** App to change the behaviour/look of the OS.(Not done yet)
